Why

This is my website in which I have spoken about all my life experiences, achievements, projects, research work, and publications. Since I love web designing, I wanted to build this from scratch and decided not to use any website builders. Instead, I started using HTML 5, Bootstrap 4, CSS 4, and Javascript to build this. This will help me in mastering my programming skills and thinking out of the box. Sometimes I had to critique my work "Because designing ain’t easy".

When

I started building this website on August 20, 2020. In web design, new designs, layouts, and icons keep updating. So this is an ongoing project. I am the only one involved in building it. It will take a lot of time to build it because all the documentation, research, and learning will be done only by me. So I will be regularly updating the website with extended features.

How

I am using HTML 5 to build the frontend, CSS 4 for the styling, Bootstrap 4 for the responsiveness of the website, and Javascript for the animations. I am using Brackets IDE as my code editor. It is an amazing IDE with extended features like “live preview” on a browser, auto code completion, syntax highlighting, and many more. I have also referred to some concepts and code snippets from unique sources. Refer to the raw code for more details.
